# On My Way 2/3

For this challenge, you'll have to find the exit point, from a start position, in a 3D map.

`prog.pwnme.fr 7001`

Everything is the same as the first challenge.

Except this time you have to send the path to get to the (E)nd

Example:

```
000
000
000
-
000
000
S00
-
000
000
0E0
-
```

answer will be `x-;z-`
